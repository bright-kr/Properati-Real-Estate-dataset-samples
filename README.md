이 **Properati Real Estate Dataset Sample**에는 **11,961개 레코드**가 포함되어 있으며, **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `Type` | 부동산 유형입니다. 예: Casa(주택), Departamento(아파트). |
| `Name` | 부동산 매물에 대한 간단한 제목 또는 이름입니다. |
| `Estrato` | 부동산 지역의 사회경제적 분류입니다. |
| `Habitaciones` | 부동산의 침실 수입니다. |
| `Banos` | 부동산의 욕실 수입니다. |
| `M2` | 제곱미터 단위의 부동산 총면적입니다. |
| `Descripcion` | 특징 및 편의시설을 포함한 부동산의 상세 설명입니다. |
| `Precio` | 부동산의 게시 가격입니다. |
| `Currency` | 부동산 가격이 표기된 통화입니다. 예: USD, ARS. |
| `Seller_name` | 부동산을 판매하는 개인 또는 에이전시의 이름입니다. |
| `Publica_desde` | 부동산 매물이 최초로 게시된 날짜입니다. |
| `Imagen` | 부동산을 나타내는 이미지 URL 목록입니다. |
| `Características` | JSON 형식의 부동산 주요 특징 및 시설입니다. |
| `Antiguedad` | 건축 후 경과 연수 기준의 부동산 연식입니다. |
| `Provincia` | 부동산이 위치한 주(Province) 또는 주(State)입니다. |
| `Region` | 부동산의 주/프로빈스 내 특정 지역 또는 구역입니다. |
| `Address` | 부동산의 전체 주소입니다. |
| `Geolocation` | 부동산의 지리 좌표(위도 및 경도)입니다. |
| `venta_arriendo` | 매물이 매매인지 임대인지 여부를 나타냅니다. |
| `Phone` | 판매자 또는 매물 에이전시의 연락처 전화번호입니다. |
| `ID` | 부동산 매물의 고유 식별자입니다. |
| `url` | 부동산 매물 상세 페이지로 연결되는 URL 링크입니다. |
| `Seller_id` | 판매자 또는 매물 에이전시의 고유 식별자입니다. |
| `Active` | 매물 상태로, 현재 활성 또는 비활성 여부를 나타냅니다. |


## Use Cases

### 1. Real Estate Market Analysis
이 데이터셋을 활용하여 부동산 시장의 트렌드를 분석하고, 아르헨티나 및 콜롬비아의 부동산 가치와 지역 수요를 평가할 수 있습니다.

### 2. Investment Opportunities
부동산 특징, 가격, 위치를 검토하여 전략적 의사결정을 위한 잠재적 부동산 투자 기회를 식별할 수 있습니다.

### 3. Urban Development Research
도시 개발 패턴과 주택 공급 가능성에 대한 연구를 수행하여 정책 수립 및 도시 계획 이니셔티브를 지원할 수 있습니다.
